# MOJAR — تطبيق إيصال استلام BTC (نسخة ويندوز)

## الملفات
- **MOJAR.html** — التطبيق كامل في ملف واحد، يشتغل **أوفلاين بدون إنترنت** (تصدير Excel و PDF شغّالين).
- **تشغيل-MOJAR.bat** — يفتح التطبيق في **نافذة مستقلة** زي أي برنامج (بدون شريط متصفح).
- **MOJAR.ico** — أيقونة التطبيق (للاختصار).

## أبسط طريقة للاستخدام اليومي
1. حِط الملفات الثلاثة في مجلد ثابت (مثلاً `C:\MOJAR`).
2. **دوبل كليك على `تشغيل-MOJAR.bat`** → التطبيق يفتح في نافذة تطبيق مستقلة.

## يثبّت في قائمة ابدأ / شريط المهام (اختياري)
1. كليك يمين على `تشغيل-MOJAR.bat` ← **إنشاء اختصار (Create shortcut)**.
2. كليك يمين على الاختصار ← **Properties** ← زر **Change Icon** ← اختر **MOJAR.ico**.
3. اسحب الاختصار إلى **سطح المكتب** أو كليك يمين ← **Pin to Start / Pin to taskbar**.

دلوقتي عندك أيقونة MOJAR تفتح التطبيق على طول، ويشتغل بالكامل أوفلاين.

## ملاحظات
- البيانات (الإيصالات المحفوظة) بتتخزّن داخل المتصفح على نفس الجهاز.
- اعمل **نسخة احتياطية (JSON)** من تبويب المحفوظات كل فترة، وتقدر تستوردها على أي جهاز.
- رقم إيصال موجار بيتولّد تلقائيًا (يبدأ 1001)؛ رقم إيصال BTC بتكتبه بإيدك.
- لو حابب نسخة تتثبّت كـ PWA أو تتنشر كموقع، استخدم مجلد `mojar-app` (فيه شرح في README بتاعه).

> لو فتحت `MOJAR.html` بالدوبل كليك مباشرة، هيشتغل برضه — بس نافذة الـ .bat بتديك شكل تطبيق أنضف.
